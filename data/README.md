# Data

## `raw/results.csv`

International football results, 1872 → present.

- **Source:** [martj42/international_results](https://github.com/martj42/international_results)
  (the same dataset published on Kaggle as
  `martj42/international-football-results-from-1872-to-2017`, maintained
  through the present day).
- **Snapshot taken:** 2026-06-10
- **Rows:** 49,450 matches
- **Columns:** `date`, `home_team`, `away_team`, `home_score`, `away_score`,
  `tournament`, `city`, `country`, `neutral`

The snapshot is committed so the whole pipeline is reproducible without
Kaggle credentials. To refresh it:

```bash
curl -sSL -o data/raw/results.csv \
  https://raw.githubusercontent.com/martj42/international_results/master/results.csv
```

> Note: the upstream file contains scheduled future fixtures (rows past
> today's date may have null scores). The pipeline filters to played matches.

## `processed/`

Generated by `scripts/01_build_data.py`:

- `elo_history.parquet` — `(date, team, rating_pre_match, rating_post_match)`
  for every team-match from the Elo burn-in start (1993) onward.
- `team_match.parquet` — long-format team-match rows (one match = two rows)
  enriched with as-of-date Elo, used to fit the model.

## `test/qatar2022_group_stage.csv`

The 48 Qatar 2022 group-stage fixtures **with actual results**. Built from the
same raw file but quarantined here so the training/prediction pipeline never
reads actual test scores until the evaluation step.

## `external/` (optional baselines)

- `qatar2022_closing_odds.csv` — manually compiled de-vigged closing 1X2 odds
  (baseline B2). Marked N/A if unavailable.
- FiveThirtyEight 2022 SPI forecasts (baseline B3), if retrievable.
